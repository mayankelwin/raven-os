# Raven-Os: The Industrial Framework for Cross-Platform Autonomy

Welcome to **Raven-Os**, a futuristic, local-first framework designed for high-performance Web and Mobile applications. Built on the principles of **Symmetry**, **E2EE**, and **Autonomy**, Raven-Os allows you to build complex, multiplayer-ready applications without managing a separate backend infrastructure.

## 🌌 Core Philosophy

Raven-Os is designed to feel like a "Firebase that runs on your machine". It unifies the development experience across platforms:
- **Symmetry**: Write once, sync everywhere.
- **Privacy by Design**: End-to-End Encryption (AES-GCM 256) is built into the protocol.
- **Developer Autonomy**: Provisions schemas and server-side logic directly from your code.

## 🏗️ Technical Stack

- **Frontend**: React (Web) & React Native (Mobile).
- **Styling**: NativeWind (Tailwind V4 for Cross-Platform).
- **Core**: Zustand + Nexus Symmetry Engine.
- **Tooling**: Raven Hub CLI (Node.js + Esbuild).

## 🚀 Getting Started

1. **Install Dependencies**: `npm install`
2. **Launch the Hub**: `npm run dev`
3. **Open Web**: `http://localhost:3000`
4. **Open Mobile**: Scan the QR code in the terminal with the Expo Go app.
